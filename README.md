## 🔥 Project Name - KWIACIARNIOK

[![View Demo](https://img.shields.io/badge/LIVE_DEMO-Visit_Website-blue?style=for-the-badge&logo=googlechrome&logoColor=white)](https://kwiaciarniok.pl/)

WebSite project for local florist's. It features an attractive design that showcases the company's offerings. HTML5, CSS3, JavaScript and Bootstrap technologies were used.

## 📚 Key Features
* **Fully Responsive Design** – Looks great on mobile, tablet, and desktop.
* **Video-Centric Layout** – Optimized for high-quality video backgrounds and portfolios.
* **Performance Optimized** – Fast loading times for a seamless user experience.
